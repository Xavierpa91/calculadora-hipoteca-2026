# 🏠 Calculadora Hipoteca 2026

Una calculadora de hipotecas moderna y elegante, diseñada para ayudarte a planificar la compra de tu vivienda.

## ✨ Características

- **Cálculo de hipoteca completo**: Cuota mensual, intereses totales, coste total de la vivienda
- **Tipo fijo, variable o mixto**: Soporte para Euríbor + diferencial y periodo promocional
- **Obra nueva / Segunda mano**: IVA 10% automático o ITP configurable con bonificación joven/VPO/familia numerosa
- **Ahorros conjuntos**: Posibilidad de sumar los ahorros de dos personas
- **Gastos por porcentaje**: Notaría (0.3-0.5%), tasación (0.05-0.1%) y gestoría (0.1-0.2%) sobre precio vivienda
- **Simulador de amortización anticipada**: Reducir cuota o reducir plazo
- **Indicadores LTV**: Porcentaje hipotecado y porcentaje de ahorros usados
- **Tipos BCE en tiempo real**: Facilidad de depósito, operaciones principales, facilidad marginal y Euríbor 12M
- **Mejores ofertas bancarias**: Comparativa de 10 bancos españoles (fijo/variable/mixto) con logos y condiciones
- **Gastos fijos mensuales**: Luz, gas, agua, IBI, AMB, alimentación, etc.
- **Tabla comparativa**: Compara diferentes plazos de hipoteca (5-35 años)
- **Diseño responsive**: Optimizado para móvil y escritorio

- Python 3 + openpyxl (versión CLI con generación de Excel)

## 📋 Uso

1. Introduce el precio de la vivienda
2. Selecciona tipo de vivienda (obra nueva / segunda mano)
3. Añade los ahorros disponibles (de 1 o 2 personas)
4. Elige tipo de hipoteca (fijo / variable) y ajusta los parámetros
5. Haz clic en "Calcular Hipoteca"
6. Consulta las ofertas bancarias en el lateral y usa "Usar en calculadora" para comparar
7. Opcionalmente, simula una amortización anticipada o añade tus gastos fijos

## 🔧 Desarrollo local

Simplemente abre `index.html` en tu navegador. No requiere servidor ni instalación.

Para la versión Python:
```bash
pip install -r requirements.txt
python Calculadora.py
```
